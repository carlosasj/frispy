Frispy
===============================

version number: 0.0.1
author: Carlos Alberto Schneider Junior

Overview
--------

A python alternative for Frisby.JS

Installation / Usage
--------------------

To install use pip:

    $ pip install frispy


Or clone the repo:

    $ git clone https://github.com/carlosasj/frispy.git
    $ python setup.py install
